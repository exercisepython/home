Fibonacci

Code a program, that asks the user how many Fibonnaci numbers to generate and then generates them.(Fibonnaci sequence: 1, 1, 2, 3, 5, 8, 13 ...)

How many numbers to generate? 4
[1,1,2,3]

Make sure to ask the user to enter the number of numbers in the sequence to generate.

Answer

          
def fibonacci():
    num = int(input("How many numbers that generates?:"))
    i = 1
    if num == 0:
        fib = []
    elif num == 1:
        fib = [1]
    elif num == 2:
        fib = [1,1]
    elif num > 2:
        fib = [1,1]
        while i < (num - 1):
            fib.append(fib[i] + fib[i-1])
            i += 1
    return fib
print(fibonacci())

Overlapping lists

Code a program, that prints out only the items from the lists, that are present in both
list1 = [2, 4, 5, 7, 9, 45, 34]
list2 = [1, 4, 9, 6, 9, 28, 45]

Extra: The duplicates from the list should be removed.

Answer

          
list1 = [2, 4, 5, 7, 9, 45, 34]
list2 = [1, 4, 9, 6, 9, 28, 45]
result = []

for i in list1:
  if i in list2:
    result.append(i) 

print(result)

Rock Paper Scissors

Code a 2 player Rock Paper Scissors game.
Ask for player plays (using input), compare them, print out a message of congratulations to the winner.

Answer

          
calc_kind = input("Kind of calculation (+,-,*,/): ")
num1 = int(input("Number 1: "))
num2 = int(input("Number 2: "))
if calc_kind == "+":
  print(num1 + num1)
elif calc_kind == "-":
  print(num1 - num1)
elif calc_kind == "*":
  print(num1 * num1)
elif calc_kind == "/" and num2 == "0":
  print("You can't divide by 0")
elif calc_kind == "/":
  print(num1 / num1)
