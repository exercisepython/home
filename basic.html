Code a program, where the user inputs 2 numbers. Then these numbers and 5 are added and the result is printed out like this: [number a] + [number b] + 5 = [result]

Sort

Code a program, that sorts all integers from the given list and prints them out individually (Google "Python sort()" for help)
list = [7,34,23,5,12,3,78,22,56,12])

Answer

          
list = [7,34,23,5,12,3,78,22,56,12])

sort_list = list.sort()
for i in sort_list:
  print(i)

Kid or adult?

Code a program, where the user inputs his age. When he's younger than 18 it should say: "You're a kid". When he's 18 or older: "You're an adult!". Be careful that the user inputs a number bigger than 0!

Answer

          
age = input("Enter your age: ")

if int(age) < 18 and int(age) > 0:
  print("You're a kid")
if int(age) >= 18 and int(age) > 0:
  print("You're an adult")
else:
  print("Please enter a number!")

Randomizer

Code a program, where the user inputs a range of 4 numbers. These numbers should be printed out in a random order

Answer

          
num 1 = input("Enter number 1")
num 2 = input("Enter number 2")
num 3 = input("Enter number 3")
num 4 = input("Enter number 4")

list = [num1, num2, num3, num4]
list.randrange(num1, num4)
