Basics

For those, who just started



The exercises


The basics course contains exercises that are easy to solve, even foer complete beginners. After this exercises you will become more confident in the language and gain important skills for further exercises.


This course is divided into 3 sections.
First you'll have to solve some easy maths tasks with Python. This is pretty dry, yet very effective. Then there some fun exercise that create little programs that develop your general Python knowledge. At the end you'll create your own "big" program.


MATHS

a+b+5=c

Code a program, where the user inputs 2 numbers. Then these numbers and 5 are added and the result is printed out like this: [number a] + [number b] + 5 = [result]

Answer

          
num1 = input("Enter number 1: ")
num2 = input("Enter number 2: ")
result = int(num1) + int(num2) + 5
print(str(num1) + str(num2) + str(5) = result)

Sort

Code a program, that sorts all integers from the given list and prints them out individually (Google "Python sort()" for help)
list = [7,34,23,5,12,3,78,22,56,12])

Answer

          
list = [7,34,23,5,12,3,78,22,56,12])

sort_list = list.sort()
for i in sort_list:
  print(i)

Kid or adult?

Code a program, where the user answers following question:

What's your age?

When he's younger than 18 it should say: "You're a kid". When he's 18 or older: "You're an adult!". Be careful that the user inputs a number bigger than 0!

Answer

          
age = input("Enter your age: ")

if int(age) < 18 and int(age) > 0:
  print("You're a kid")
if int(age) >= 18 and int(age) > 0:
  print("You're an adult")
else:
  print("Please enter a number!")

Test paper

Code a program, where the user answers the following question:
What is 2+2?

When the user enters the right answer (4) it should print: "You're talented!".
When not: "You need to practise some maths!"

Answer

          
task = input("What is 2+2? ")
if task == int(4):
  print("You're talented!")
else:
  print("You need to practise some maths!")

FUN

Login

Code a program, where the user inputs a username and a password (the password is "Python"). When the password is right the the program should welcome the username like this:

Welcome [username]!

When not the program should say

Wrong password!

Answer

          
username = input("Username")
password = input("Password")
if password == "Python":
  print("Welcome " + str(username))
else:
  print("Wrong password!")

Birthday Calendar

Code a program, where the 3 users can input their names and birthdays (one by one)

Name:
Birthday:

After this a list with 3 names and birthdays should appear on the screen (use a loop!)

[name1]: [birthday 1]
etc

Answer

          
name1 = input("Name 1: ")
birth1 = input("Birthday 1: ")
name2 = input("Name 2: ")
birth2 = input("Birthday 2: ")
name3 = input("Name 3: ")
birth3 = input("Birthday 3: ")

all = [name1, birth1, name2, birth2, name3, birth3]

for i in all:
  print(i)

What's the highest mountain on earth? [Mount Everest]

APPLICATION

Calculator

Code an application, where the user inputs what kind of calculation he wants to make (+,-,*,/). Then he inputs 2 numbers and the result has to be calculated. Make sure, that he can't divide by 0.

Kind of calculation (+,-,*,/):
Number 1:
Number 2:
Result:

Answer

          
calc_kind = input("Kind of calculation (+,-,*,/): ")
num1 = int(input("Number 1: "))
num2 = int(input("Number 2: "))
if calc_kind == "+":
  print(num1 + num1)
elif calc_kind == "-":
  print(num1 - num1)
elif calc_kind == "*":
  print(num1 * num1)
elif calc_kind == "/" and num2 == "0":
  print("You can't divide by 0")
elif calc_kind == "/":
  print(num1 / num1)
